# Space-booking
Space-booking은 공간대여 시스템으로, SpringBoot 를 이용해 개발했습니다.
  
## 🖥️ 프로젝트 소개
Space-booking은 HyejinShin의 개인 토이 프로젝트입니다.
  
## 🕰️ 개발 기간
* 25.10.10 ~ 25.10.

### ⚙️ 개발 환경
- **Language** : Java 21  
- **Framework** : Spring Boot (3.5.6)  
- **Database** : MySQL
- **ORM** : JPA  
- **Frontend** : JSP, HTML, CSS, JavaScript  
- **Build** : Gradle

### 🖥️ 서버 환경
- Apache Tomcat 9.0.65
- JDK 21
